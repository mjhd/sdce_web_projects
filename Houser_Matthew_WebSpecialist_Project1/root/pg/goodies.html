Awesome Development Tools:

Resources

  • W3 Schools

    Tutorials, explanations, detailed articles... Everything you need to get started!

  • Stack Overflow

    A free help forum loaded with pros. Ask a question, describe a problem, or view solutions to issues similar to yours

  • Smashing Magazine

    Smashing Magazine is an online publication that specializes in a design first approach to the web. Its very current and forward thinking, with fantastic articles, ideas, assets & examples.

  • jsFiddle

    A sandbox to build small elements before you implement. The editor has HTML, CSS, and JS sections to be used and is equipped with several built in JavaScript libraries to test with.

Training

  • CodeCademy

    A free to all online training website, nicely categorized into sections and oriented around constant practice. Its code challenges offer a sandboxed editor, code hints, and checks all your work!

  • Team Treehouse

    Treehouse is quickly becoming a universally recognized leader in online training outside the traditional education pathway. It has some of the most well thought out, organized content I have ever seen. Requires a monthly subscription

  • Mozilla's P2PU

    From the Mozilla Foundation comes an oddball contender with P2PU. A forum based community driven training resource (free to all) that emphasizes interacting with others learning in your current content track. It takes a bit of getting used to but is actually quite thorough.

  • Learnable

    Learnable is another subscription based resource. While its basics are pretty lax, the specific and more advanced training with specific frameworks, API's, and techniques (like advanced AJAX). This option is most definitely not for beginners but for more advanced users trying to expand there skillset.

Templates, Bootstraps, & Bases

  • HTML5 Initializer

    Initializer is a fantastic little tool that creates a custom, blank, HTML5 template to build on based on your custom specifications. It includes custom classes, responsive defaults, and Bootstrap as well as modernizr for backwards compatibility, or fully customizable HTML5 Polyfills.

  • jQuery4u (jQuery mobile bootstraps page)

    jQuery4u is a jQuery centered page with templates, plugins, bootstraps and information. At this point I should probably explain the Twitter Bootstrap is a name for a specific bootstrap. A bootstrap is a preconfigured environment with UI, framework, and data handling/persistence capabilities built in. The link leads specifically to the jQuery Mobile bootstrap/template page.

  • HTML5 UP

    HTML5 UP is a repository of responsive HTML5 templates based on a modified version of the 960 grid framework.

  • Aria Templates

    Aria Templates are highly scalable and customizable web app templates with a frontend/backend ready JS core that works beautifully with any backend development environment, MVC, language, CMS, and client side interactive UI or DOM manipulation library. It provides only the intermediary framework to interface front and back with easy analytics and app monitoring integration.

Validators and Minifiers

  • W3C Validation Service

      THE
    Validator. The W3C Validator is the absolute go to and most important as the W3C maintains the most widely regarded "official" web standards. ALWAYS use this validator for final validation

  • Validome

    Validome is another great validator that offers more languages and standards than the W3C validator does. This should ONLY be used to generally guide practices not thoroughly covered by the W3C validator and should ALWAYS be double checked

  • Google Closure Compiler

    Google's Closure Compiler is a multi purpose JavaScript minifier that will remove blank spaces and simple errors or redundancies. It also checks syntax, searches for unused or valueless variables and blank functions, producing what its site refers to as "better JavaScript" - I agree! Generally any utility offered by Google is top notch and they really outdid themselves with Closure Compiler.

  • CompressMyCode

    CompressMyCode is not nearly as functional or useful as Google Closure Compiler for JavaScript, however it can minify HTML and CSS for web apps (or websites though I don't recommend doing so) which is particularly useful for mobile apps and AJAX assets.

Web Based Graphic Suites

  • ipiccy

    Ipiccy is an online image editor and touch up suite. Its completely free and pretty robust for an online editor.

  • Sumo Paint

    Sumo Paint is a free MSpaint style graphics creation suit online. It boasts layers, vector graphics, custom brushes, and vector shape tools. Also very good considering its online!

  • Create Graph

    A very simple graph creator designed for kids, but great for creating quick & easy infographics.

  • bgmaker

    The worlds simplest pixel by pixel tiling background maker. Make your graphic and click download, its that easy!
